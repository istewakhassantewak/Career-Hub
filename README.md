# 💼 Career Hub

A modern job listing and application tracking web application built with **React.js**, **React Router**, and **Tailwind CSS**. Users can browse jobs, view details, and apply for positions with applied jobs stored locally using **localStorage**.

---

## 🚀 Live Demo

🔗 Live Site: https://career-hub-smoky.vercel.app

---

## 📂 GitHub Repository

🔗 Repository Link: https://github.com/istewakhassantewak/Career-Hub

---

## ✨ Features

- 💼 Browse available job listings
- 📄 View detailed job descriptions
- 📌 Apply to jobs with one click
- 💾 Applied jobs stored in localStorage
- 🔍 Persistent applied job tracking
- ⚛️ Built with React + Hooks
- 🌐 Client-side routing using React Router
- 🎨 Modern UI with Tailwind CSS + DaisyUI
- 📱 Fully responsive design

## 📦 Installation & Setup

Clone and run locally:

```bash
# Clone repository
git clone https://github.com/istewakhassantewak/Career-Hub.git

# Move into project folder
cd Career-Hub

# Install dependencies
npm install

# Start development server
npm run dev
🖥️ Build for Production
npm run build
